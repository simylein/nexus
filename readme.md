## nexus home

a simple web application which acts as a transceiver for lora

### compile the source code

for development

```sh
make develop
```

for production

```sh
make release
```

### initialize the database

```sh
./nexus --init
```

### populate the database

```sh
./nexus --seed
```

### start the application

```sh
./nexus
```

and enjoy ;^)

### configure the application

check out the available command line flags

```sh
./nexus --help
```
